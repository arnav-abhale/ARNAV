# 📌 Featured Project

## Aarogya Connect

A full-stack healthcare awareness platform designed to help NGOs promote healthcare camps and allow underserved communities to discover camps and book slots online.

### Key Features

* JWT-based Authentication
* Slot Booking System
* Multilingual Support
* Responsive User Interface
* NGO Healthcare Camp Awareness

### Tech Stack

React.js • Next.js • Node.js • Express.js • PostgreSQL • Tailwind CSS

🔗 Live Demo: https://healthcarewebappuidesign.vercel.app/
🔗 GitHub: https://github.com/Harshp006/aarogyaconnect
